# learn-mobx

mobx 练习

## 项目初始化

这里使用 umi@4 做项目初始化, 环境配置如下

- node@16
- pnpm@7.26.3
- mobx@6
- umi@4

```bash
# 初始化项目
pnpm dlx create-umi@latest --no-install
pnpm i

# 开发
pnpm dev

# 安装 mobx mobx-react
pnpm i mobx mobx-react
```
